
# Infrared Converter

A tool for converting infrared files.

## Project Structure

Infrared converter/ ├── input/ # Place input files here (contents ignored by git) ├── model/ # Model files go here (contents ignored by git) ├── output/ # Output files go here (contents ignored by git) └── test.py # Main test script


## Usage

1. Place your input files in the `input/` folder
2. Ensure the required model files are in the `model/` folder
3. Run `test.py`
4. Results will be saved to the `output/` folder

## Notes

- The `input/`, `model/`, and `output/` folders are tracked by git, but their contents are ignored via `.gitignore`.
- Place any required files locally after cloning.
